# Ruboty::Rss

## Installation

In Gemfile:

```
gem 'ruboty'
gem 'ruboty-rss'
```

If you would like to set interval of checking feeds, set `RUBOTY_RSS_INTERVAL` envvar in seconds.

## Usage

See `ruboty help`.
